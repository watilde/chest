chest.js

The easy metafile manager

Why use a metafile manager?

Using npm.

Install supported metafiles

Type once, all initial install done


$ chest install

If there is the following files in the chest, run initial install command

File name Command to run
package.json $ npm install
bower.json $ bower install
component.json $ component install
