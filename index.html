chest.js

The easy metafile manager

Why use a metafile manager?

Reduce your costs manage metafiles:
npm, grunt, bower, gulp, component, jshint, editorconfig, travis, saucelab and so on,
Solutions like the above for helping the development of front-end continues to grow now. In the same way, metafiles like also continues to grow. Outlook of the root directory isn't only become worse, but also the number of commands that you run after the git clone will become to complicated.

Install supported metafiles

Type once, Run all initial install: 
$ chest install

If there is the following files in the chest, run initial install command.

File name Command to run
package.json $ npm install
bower.json $ bower install
component.json $ component install
